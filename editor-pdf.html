Como usar:
1. Importe o PDF.
2. Clique sobre o campo a editar — uma caixa será criada.
3. Digite o texto novo na caixa (ou no painel ao lado).
4. Arraste para mover, alça ● para redimensionar.
5. Delete remove a caixa selecionada.
6. Exporte para baixar o PDF editado.
